
## 获取CZR最新的价格


- 方式 ：GET
- URL : `https://api.etherscan.io/api`
- 参数
    ```
    {
        module      : account ,
        action      : ethprice ,
        apikey      : YourApiKeyToken
    }
    ```
- 结果


```
https://api.etherscan.io/api?
    module=stats&
    action=ethprice&
    apikey=YourApiKeyToken

```
